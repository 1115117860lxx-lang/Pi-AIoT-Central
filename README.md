# Pi-AIoT-Central

#基于 Raspberry Pi 5 + NPU 的边缘计算私有化 AIoT 中控系统（架构篇）

#功能特性 (Features)：
✅ 离线大模型大脑： 集成 deepseek-r1:8b，无需联网即可理解自然语言。
✅ 数字孪生架构： 前后端分离设计，具备完整的 Web 仿真控制台。
✅ 结构化指令引擎： 独创 System Prompt，将自然语言 100% 转化为 JSON 控制协议。
